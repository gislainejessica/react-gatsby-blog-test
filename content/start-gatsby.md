## Utilizando Gatsby, como fazer um blog em Gtasby?
Instalar gatsby-cli globalmente para criar os projetos
`npm install -g gatsby-cli`
criar um site pra colocar tuas ideias no ar
`gatsby new gatsby-site`

`cd gatsby-site`

`code .`

`gatsby develop`
